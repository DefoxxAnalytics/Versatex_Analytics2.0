Upload Procurement Data

Upload a CSV file containing procurement transactions. Data will be imported into the database.

Required CSV Columns:

  • supplier - Supplier name (will be created if new)
  • category - Category name (will be created if new)
  • amount - Transaction amount (numeric)
  • date - Transaction date (YYYY-MM-DD or MM/DD/YYYY)

Optional Columns:

  • description, subcategory, location, fiscal_year, spend_band, payment_method, invoice_number
